# Handmade Shop Website

## Project Description
This project is a simple static website created as part of a web development assignment.  
The goal of the project was to practice GitHub workflow, branch management, Pull Requests, and responsibility distribution in a development environment.

The website consists of three separate HTML pages:
- Home page
- Services
- Contact 

Navigation works between all pages, and the layout is built using Bootstrap framework.

---

## Technologies Used
- HTML5
- CSS
- Bootstrap 5
- Git
- GitHub
